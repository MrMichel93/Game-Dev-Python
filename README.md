# Game Development with Python - AP Computer Science Principles Course

A comprehensive, 10-module introductory course on game development using Python and Pygame Zero, fully aligned with AP Computer Science Principles curriculum.

## ğŸ“š Course Modules

### Module 1: Variables and Game State
**Big Idea:** Data (DAT)  
Learn how variables represent and manage game state, from player position to score tracking.

### Module 2: Conditionals and Game Logic
**Big Idea:** Algorithms and Programming (AAP)  
Apply conditional statements to control game behavior and create dynamic decision-making systems.

### Module 3: Loops and Animation
**Big Idea:** Algorithms and Programming (AAP)  
Use iteration to create repeated actions, animations, and process collections of game objects.

### Module 4: Lists and Sprite Management
**Big Idea:** Data (DAT)  
Manage collections of game objects efficiently using lists and list operations.

### Module 5: Functions and Abstraction
**Big Idea:** Algorithms and Programming (AAP)  
Organize code into reusable functions and understand abstraction as a complexity management tool.

### Module 6: Dictionaries and Game Data Structures
**Big Idea:** Data (DAT)  
Use dictionaries to organize complex game data and represent game entities.

### Module 7: Event Handling and User Input
**Big Idea:** Computer Systems and Networks (CSN)  
Implement responsive controls and understand event-driven programming.

### Module 8: Collision Detection and Game Physics
**Big Idea:** Algorithms and Programming (AAP)  
Implement collision detection algorithms and apply basic physics principles.

### Module 9: Game Design and Creative Development
**Big Idea:** Creative Development (CRD)  
Apply iterative design processes and learn how collaboration enhances game development.

### Module 10: Game Impact and Ethics
**Big Idea:** Impact of Computing (IOC)  
Examine the societal implications of games and learn to design accessible, ethical games.

## ğŸš€ Getting Started

### Prerequisites
- Python 3.8 or higher
- Basic computer literacy
- Enthusiasm for games and problem-solving
- No prior programming experience required

### Installation

1. **Install Python**
   - Download from [python.org](https://www.python.org/downloads/)
   - Ensure Python is added to your PATH

2. **Install Pygame Zero**
   ```bash
   pip install pgzero
   ```

3. **Clone this repository**
   ```bash
   git clone https://github.com/MrMichel93/Game-Dev-Python.git
   cd Game-Dev-Python
   ```

4. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

### Running Examples

Navigate to any module folder and run the example files:
```bash
cd modules/module_01
pgzrun example_game_state.py
```
